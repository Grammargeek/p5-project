Design Your Christmas Tree!

This is a Christmas interactive project. You can use this webpage to make your own Christmas tree! Type in the colors in the numbered order for the best results, if the name of the color is two words, like "dark grey" write it with no spaces, like "darkgrey". Don't worry about the triangles showing through the layers, it will go away once you have finished with the colors.
